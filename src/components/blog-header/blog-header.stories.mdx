# Blog Header
The blog header consists of a `kicker`, `headline`, a `byline` to note the author & time. The content it receices through the `children` prop acts as the `lead` text for the following blog content.

## Usage
The `BlogHeader` should only be used within a blog article which is wrapped by ab `<article></article>` to ensure a proper semantic markup.
