# Checkbox

Checkboxes are used for a list of options where the user may select multiple options, including all or none.

Besides the `label` it receives props of the type `UseControllerProps`. The API of this component is based on the [controlled approach of react-hook-form](https://react-hook-form.com/api/usecontroller).
